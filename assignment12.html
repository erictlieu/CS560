A12: Black Hat White Hat

  1. Black Hat
    • Chart contains biased title
    • Visualization is area chart which does not mean anything with temperature
    • Temperature axis label goes from 0 to 100 when it should be from min to max of data
    • Temperature axis missing units (converted into Fahrenheit)
    • No reference to data source
  2. White Hat
    • Line chart is easy to understand when dealing with time series temperature data
    • Unbiased title
    • Temperature axis is properly labeled and scaled correctly
    • Informing the user only January data is used
    • Annotations help the user focus on the topic
    • There is a reference to data source
