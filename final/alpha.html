Interactive Data Visualization with Movie Reviews: Alpha Release

By Eric Lieu

Overview:

This project will be an interactive data visualization dashboard using a dataset of movies and movie reviews.

There will be a node-link diagram with movies as nodes and user reviews of same movies as edges.

On the side would be a chord diagram displaying all genres and movies sharing those genres.

Below the chord diagram would be a rating histogram.
